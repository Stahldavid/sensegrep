# Support

## Getting Help

- Usage questions: open a GitHub Discussion or issue with context and repro steps.
- Bug reports: use the bug report issue template.
- Feature requests: use the feature request issue template.
- Security issues: follow `SECURITY.md` and use private reporting.

## Before Opening an Issue

- Check existing issues and discussions first.
- Include your OS, Node version, and sensegrep version.
- Share a minimal reproduction when possible.

## Project Status

sensegrep is actively maintained. Priorities are reliability, search quality, and MCP/CLI usability.
